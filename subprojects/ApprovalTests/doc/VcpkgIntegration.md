## Using vcpkg to obtain ApprovalTests.cpp

Approval Tests is available via vcpkg, since v.10.9.1

Windows:

```bash
.\vcpkg install approval-tests-cpp
```

Linux/Mac:

```bash
vcpkg install approval-tests-cpp
```
